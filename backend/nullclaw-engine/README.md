# NullClaw Engine

This directory is scaffolded to contain the NullClaw engine.
The user requested cloning the engine from GitHub, but because this is a monorepo setup running in an environment without standard `git`/`mkdir` bash execution available via tools, we mock the directory structure here.

In a normal environment, this would be populated via:
`git clone https://github.com/nullclaw/nullclaw backend/nullclaw-engine`

The `backend/gateway` is built to wrap this logic anyway.
